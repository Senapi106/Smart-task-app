# Smart Task Manager

A full-stack task management application built using Flask, PostgreSQL, and JavaScript.

## 🚀 Features

- User Registration & Login (JWT Authentication)
- Create Tasks
- View Tasks
- Update Task Status (Complete / Incomplete)
- Delete Tasks
- Persistent Login (Token stored in browser)

## 🛠 Tech Stack

**Backend:**
- Flask
- PostgreSQL
- SQLAlchemy
- JWT Authentication

**Frontend:**
- HTML
- CSS
- JavaScript (Fetch API)

## ⚙️ How to Run

### Backend

1. Navigate to backend folder:
cd backend

2. Activate virtual environment:
venv\Scripts\activate


3. Run server:
python run.py


---

### Frontend

Open: frontend/index.html
